# 🚀 Mis Ejercicios de Java - DAW1

Aquí voy subiendo algunos ejercicios que realizamos en 1º del ciclo de **DAW** (Desarrollo de Aplicaciones Web). Lo uso para repasar y tener ejemplos de código a mano.

---

## 📋 Índice de Ejercicios

* **ArrayInt.java**: Operaciones básicas con arrays normales (`int[]`). Sumar, buscar el mayor y el menor.
* **ArrayInteger.java**: Uso de listas dinámicas (`ArrayList`) y objetos. Aquí probé el método `getFirst()` de Java 21.
